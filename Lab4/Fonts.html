Font colour

The HTML "style" attribute is used to add styles to an element, such as color, font, size, and more.

The fonts can be written in various colours such as

  • Red
  • Blue
  • Yellow
  • Green
  • Aquamarine
  • Orange

    • Font Styles

    There are various fonts styles that can be used in the creation of a web page. Using the attribute "font-family" we can change the font styles.

    Some various examples for Font styles is

  • Courier New
  • Times New Roman
  • Verdana
  • Cursive
